# Project D.A.R.T.A. Website

This is the official website for Project D.A.R.T.A. by Team Lego Dynamics.

## Overview

Project D.A.R.T.A. is an initiative to make archaeology more accessible and inclusive through innovative technology, including drones, ground-penetrating radar, mixed reality, and AI analysis.

## Features

- Modern, responsive design using [Bootstrap 5](https://getbootstrap.com/)
- Based on the [Start Bootstrap "Agency" template](https://startbootstrap.com/theme/agency)
- Parallax header effect for a dynamic landing experience
- Sections for project goals, features, pricing, team, and contact
- Accessible and inclusive design focus

## Getting Started

### Prerequisites

- A web server or local environment to serve static files (e.g., [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) for VS Code)
- Node.js and npm (optional, for dependency management)

### Installation

1. Clone or download this repository.
2. Ensure all dependencies in `node_modules` are installed (if using npm).
3. Open `index.html` in your browser, or use a local server for best results.

### Folder Structure

- `index.html` — Main website file
- `assets/` — Images, icons, and other static assets
- `css/` — Custom and Bootstrap CSS
- `js/` — JavaScript files
- `node_modules/startbootstrap-agency/` — Start Bootstrap Agency template files (everything in `dist/` has been moved to root for simplicity)

## Customization

- The site uses Bootstrap 5 and the Start Bootstrap Agency template for layout and components.
- Parallax effect and custom styles are added in `index.html`.
- Update content, images, and styles as needed for your project.
